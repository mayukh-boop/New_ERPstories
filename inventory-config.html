Lot & Bin Tracking
Control how deeply individual material batches and physical locations are tracked.
Lot Tracking
Every GRN receipt creates a lot number (LOT-YYYY-NNNN). All stock movements carry the lot ID. Required for ageing reports and FIFO allocation.
Bin Management Tier 3
Requires Lot Tracking ON. Each lot is assigned to a physical bin (FAB-A-R03S2-B04). Pick suggestions on Issue Notes. Bin-by-bin stock counts.
Hard-Enforce Bin Capacity
If ON: assignment exceeding max capacity is blocked with an error. If OFF: warning only.
Allow Mixed Items in One Bin
Default OFF — one bin holds one item type only. Turn ON only for cage-style storage where mixing is physically unavoidable.
warehouse
Warehouse
Single vs. multi-warehouse operation. Inter-warehouse transfers are only available when multi-warehouse is enabled.
Multi-Warehouse Tier 3
When OFF: all movements use the default warehouse and the warehouse selector is hidden on all documents. When ON: warehouse is required on every document and inter-warehouse transfer is enabled.
Default Warehouse
Used for all movements when multi-warehouse is OFF.
verified
QC Mode
Controls how quality inspection is handled at GRN receipt. Aligns with tickets NE-14A (Inline) and NE-14B (QC Hold).
Inspection Mode
INLINE — Accept/reject quantities directly on the GRN line at the dock. No separate QC step.
QC_HOLD — Full received qty goes to QC Hold area. QC Inspector reviews separately on the QC Inspection page.
BOTH — Each GRN line has a mode toggle: inspector chooses inline or hold per line.
receipt_long
BOM & Reservation
Controls the Style BOM approval workflow and how material is reserved against production orders.
BOM Approval Required
When ON: a Style BOM must be ADMIN-approved (DRAFT → ACTIVE) before it can drive reservations or Cutting Issue Notes. When OFF: BOM becomes ACTIVE immediately on save.
Reservation Mode
MANUAL — Planner explicitly reserves free stock via the Reserve action. "Reserve" button visible on production order detail.
AUTO — System auto-reserves from free stock (FIFO) when a Production Order is created. Reserve button hidden on UI.
Auto-Reserve on Production Order Creation Tier 3
Only applicable when Reservation Mode = AUTO. Triggers reservation immediately when a PO is created — no manual step needed.
Allow Negative Stock
When OFF (recommended): Issue Note is blocked if issued qty would exceed available balance. When ON: warning is shown but movement posts regardless. Useful during system go-live when opening balances are incomplete.
warning Enabling Allow Negative Stock means issued quantities are not validated against available balance. Only enable during data migration or opening balance setup — disable before going live.

Issue Notes
Controls the Cutting Issue Note approval workflow before stock physically leaves the store.
Issue Note Approval Required
When ON: Store Keeper creates DRAFT → submits for approval → Supervisor approves → issued. When OFF: Store Keeper creates and issues directly (DRAFT → ISSUED), no approval step.

payments
Costing Method
Controls whether inventory value is computed and shown on stock views and ageing reports. Unit cost is taken from the GRN line at receipt time.
Costing Method
NONE — Qty only. No value column on any report. Fastest and simplest.
FIFO_COST — First-in-first-out costing. Value of stock = cost of oldest lots first.
WEIGHTED_AVG — Weighted average unit cost recalculated on every receipt and write-off. Tier 3 recommended
info When costing method is not NONE, the Ageing Report shows Value at Risk (balance qty × unit cost) and the Stock Dashboard shows Unallocated Value.
